CS3050 Group12 Final Project
--------
This is a Netbeans project.  
The only needed argument is the maze text file.  
Output will be on standard out.  On a successful run it will print 4 mazes, all labeled:
1. The original maze
2. The maze with the path from S to E marked by '.....'
3. The maze cleared of S, E, and S's path
4. The maze with the path from F to L marked by '......'
